The ratio I decided to use for this project was the ratio between county population and the entire population of Oregon. I thought that this provided a good indication for the percentage of people that lived in each county in the state and could be classified further more easily. It also is good variation as opposed to total people in each county as it made each county bound to one standard number. As far as the classifications, I chose equal interval, quantile, and standard deviation. Equal interval divides the range of attribute values into equally sized classes, which are determined by the user. In this case I chose seven classes. Some pros of using equal interval is that it is efficient for continous data sets, and also creates a legend that is easily read by any audience. Cons are that most of the data set will usually end up taking one or two classes leaving it hard to see varation in the data. The next classification used was quantile. This classification places equal numbers of observations into each class. Pros of quantile classification is that it excels at emphasizing the relative position of the data values. However, in this method values placed in the same class can be extremely different and not that related to one another. The last classification method used was satndard deviation. This method places values into classes based on how many standard deviations away they are from the mean. Pros to standard deviation include it is very useful for data with normal distribution, but if the data is irregular it may be innefective.
